# プロダクト概要

GROWI MCP Server - AIアシスタントとGROWI Wikiコンテンツを接続するModel Context Protocol (MCP) サーバー。LLMがナレッジベースのコンテンツを検索・取得・管理し、コンテキストを考慮した応答を可能にする。

## コア機能

- **Wikiコンテンツアクセス**: MCPツールを通じてGROWIインスタンスからページを検索・取得
- **ページ管理**: 完全なCRUD操作（作成、読み取り、更新、削除、名前変更、複製）
- **マルチアプリ対応**: 名前付き設定で複数のGROWIインスタンスに同時接続
- **コンテンツ整理**: タグ管理、共有リンク、リビジョン履歴、コメント

## 想定ユースケース

- **ナレッジ取得**: AIアシスタントが組織のWikiから正確で最新の情報を取得
- **コンテンツ自動化**: AIワークフローによるプログラマティックなページ作成・更新
- **マルチ環境運用**: 本番、ステージング、開発環境のGROWIインスタンス間でのコンテンツ管理

## 価値提案

- **MCPネイティブ**: Model Context Protocolに基づくシームレスなAIツール統合
- **TypeScript SDK統合**: 公式GROWI SDKを活用した信頼性の高いAPI連携
- **柔軟な設定**: 番号付き環境変数による環境ベースのマルチアプリ設定
- **npm配布**: `@growi/mcp-server`としてnpxで簡単にインストール可能
